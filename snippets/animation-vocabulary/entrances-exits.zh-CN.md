# 进入与退出动画

用于描述元素如何出现、消失、进入界面或离开界面。

- **Fade in / Fade out（淡入 / 淡出）**：元素通过透明度变化逐渐出现或消失。
- **Slide in（滑入）**：元素从可视区域外滑入界面，可以来自左、右、上、下等方向。
- **Scale in（缩放进入）**：元素从较小尺寸放大到最终尺寸，通常会配合透明度变化。
- **Pop in（弹入）**：元素出现时带有轻微过冲或回弹，最后再稳定到目标状态。
- **Reveal（揭示）**：内容被逐步露出，常通过 `clip-path`、遮罩或裁切容器实现。
- **Enter / Exit（进入 / 退出）**：元素被添加到界面或从界面移除时播放的动画。
