# Transitions Between States

Use these terms to describe how one state, view, layout, or element visually connects to another.

- **Crossfade**: One element fades out while another fades in at the same location.
- **Continuity transition**: A transition that preserves orientation by visually linking the before and after states.
- **Morph**: One shape or element smoothly transforms into another.
- **Shared element transition**: The same element appears to travel and transform between two layouts or views.
- **Layout animation**: Size or position changes animate into place instead of snapping instantly.
- **Accordion / Collapse**: A section expands or collapses its height to show or hide content.
- **Direction-aware transition**: Navigation motion changes direction based on whether the user is moving forward or backward.
