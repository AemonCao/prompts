# 循环、质感与性能

用于描述环境动效、细节质感，以及影响动画流畅度的性能因素。

## 循环与环境动效

- **Marquee（跑马灯）**：文字或内容持续循环滚动。
- **Loop（循环）**：动画重复播放，可以是固定次数，也可以是无限循环。
- **Alternate / Yoyo（交替 / 往返）**：循环动画先正向播放，再反向播放，而不是直接跳回起点。
- **Orbit（环绕）**：一个元素围绕另一个元素或某个中心点持续运动。
- **Pulse（脉冲）**：尺寸或透明度轻微、重复地变化，用于吸引注意。
- **Float（漂浮）**：元素持续轻柔地上下移动，营造轻盈感。
- **Idle animation（闲置动画）**：元素等待交互时播放的细微运动。

## 质感与效果

- **Blur（模糊）**：通过滤镜柔化元素，或掩盖细小视觉瑕疵。
- **Clip-path（裁切路径）**：用裁切形状实现揭示、遮罩或对比效果。
- **Mask（遮罩）**：用形状或渐变隐藏、露出元素的部分区域，边缘可以比 `clip-path` 更柔和。
- **Before / after slider（前后对比滑杆）**：通过可拖动分隔线在两张叠放图片之间进行对比。
- **Line drawing（线条绘制）**：SVG 路径像被画笔逐步描出一样出现。
- **Text morph（文字形变）**：文本变化时，字符或字形带有动画过渡。
- **Skeleton / Shimmer（骨架屏 / 微光）**：加载时展示占位结构，并带有移动光泽。
- **Number ticker（数字滚动）**：数字滚动或递增到新的数值。
- **Tabular numbers（等宽数字）**：数字宽度一致，避免计数器或计时器在变化时左右跳动。
- **Typewriter（打字机效果）**：文字像被逐字输入一样出现。

## 性能

- **Frame rate / FPS（帧率）**：每秒绘制的帧数，`60fps` 通常是流畅动画的基础目标。
- **Jank（卡顿）**：因为浏览器没有及时绘制帧而产生的可见停顿。
- **Dropped frame（掉帧）**：没有在规定时间内完成绘制的一帧。
- **Compositing（合成）**：让 GPU 在独立图层上移动或淡化元素，避免频繁重新布局和重绘。
- **will-change（预声明变化）**：CSS 提示，用来告诉浏览器某个元素即将发生动画变化。
- **Layout thrashing（布局抖动）**：反复触发布局计算造成卡顿，常见于动画化 `width`、`height`、`top`、`left` 等属性。
