# 状态与视图过渡

用于描述一个状态、视图、布局或元素如何视觉上连接到另一个状态。

- **Crossfade（交叉淡化）**：一个元素淡出，同时另一个元素在同一位置淡入。
- **Continuity transition（连续性过渡）**：通过视觉连接前后状态，让用户保持方向感和空间感。
- **Morph（形变）**：一个形状或元素平滑地转变成另一个形状或元素。
- **Shared element transition（共享元素过渡）**：同一个元素在两个布局或视图之间移动并变形，例如缩略图展开成详情卡片。
- **Layout animation（布局动画）**：元素尺寸或位置变化时平滑过渡到新位置，而不是瞬间跳变。
- **Accordion / Collapse（手风琴 / 折叠）**：区域通过高度展开或收起，以显示或隐藏内容。
- **Direction-aware transition（方向感知过渡）**：根据用户前进或返回的方向，使用相反或对应方向的动效。
