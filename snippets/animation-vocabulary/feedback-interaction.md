# Feedback & Interaction

Use these terms to describe motion that responds to user actions.

- **Hover effect**: A visual change when the pointer moves over an element.
- **Press / Tap feedback**: A small response, often a slight scale-down, when an element is clicked or tapped.
- **Hold to confirm**: A progress animation that completes only while the user keeps holding.
- **Drag**: Moving an element by grabbing it, often with momentum on release.
- **Drag to reorder**: Rearranging list items by dragging while nearby items shift aside.
- **Swipe to dismiss**: Dragging an element off-screen to close or remove it.
- **Rubber-banding**: Resistance and snap-back when dragging beyond a boundary.
- **Shake / Wiggle**: A quick side-to-side motion that signals an error or rejected action.
- **Ripple**: A circle or wave expands from the tap point to confirm input.
