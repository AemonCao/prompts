# Movement & Transforms

Use these terms to describe how an element changes position, size, angle, depth, or transform origin.

- **Translate**: Moves an element along the X or Y axis.
- **Scale**: Makes an element larger or smaller.
- **Rotate**: Turns an element around a point.
- **Skew**: Slants an element along the X or Y axis, creating a sheared shape.
- **3D tilt / Flip**: Rotates an element in 3D space with `rotateX` or `rotateY`.
- **Perspective**: Controls the apparent strength of depth in a 3D transform.
- **Transform origin**: The anchor point that scaling or rotation happens around.
- **Origin-aware animation**: Motion that appears to grow from the trigger or source element instead of from the animated element's center.
