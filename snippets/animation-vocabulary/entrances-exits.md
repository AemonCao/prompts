# Entrances & Exits

Use these terms to describe how interface elements appear, disappear, enter, or leave the screen.

- **Fade in / Fade out**: An element appears or disappears by changing opacity.
- **Slide in**: An element enters from outside the visible area, usually from the left, right, top, or bottom.
- **Scale in**: An element grows from a smaller size into its final size, often paired with opacity.
- **Pop in**: An element appears with a small overshoot or bounce before settling.
- **Reveal**: Content is uncovered progressively, often with `clip-path`, a mask, or a clipping container.
- **Enter / Exit**: The motion played when an element is added to or removed from the interface.
