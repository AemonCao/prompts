# 图像取证式复刻生图 Prompt

请把这张图片当成犯罪现场。

你的任务不是欣赏它，而是彻底逆向拆解它的生成逻辑。

你必须分析：

- 摄影师站在哪里
- 镜头为什么这样变形
- 为什么这里过曝
- 为什么这里失焦
- 为什么主体不在中心
- 为什么看起来“真实”
- 为什么像“随手拍”
- 为什么有廉价感
- 为什么像旧互联网图片

然后生成一个：

> “能够最大程度复刻原图”的 AI 生图 Prompt。

要求：

不要文学化描述。
不要写感受。
不要写“一个很酷的人”。

必须全部转换成：

可被 AI 绘图模型理解的视觉语言。

重点分析：

- 镜头缺陷
- 手机算法痕迹
- HDR 痕迹
- JPEG 压缩
- 感光噪点
- 自动白平衡错误
- 廉价 CMOS 质感
- 社交媒体压缩感
- TikTok / 微博 / 小红书 / Instagram 图片风格
- 低质量运动模糊
- 非专业构图
- 偷拍感
- 监控感
- 意外抓拍感

最终输出：

直接给出可用于 AI 生图的 Prompt。
不要解释。
不要分点。
不要说“这张图片里”。

输出结构：

```text
[Main Prompt]

[Negative Prompt]

[Flux Version]

[Midjourney Version]

[Cinematic Version]
```
